# Política de Segurança (SSOT_PUBLIC)

Este repositório é **público** e contém apenas material **SSOT_PUBLIC** (sanitizado).

## Como reportar um problema de segurança
- Abra uma **Issue** com o mínimo de detalhes possível
- Marque como **security**
- **Não** anexe dados reais (tokens, chaves, cookies, payloads, logs brutos)

## O que NÃO deve aparecer aqui
- Credenciais (tokens, chaves, cookies)
- Logs com payload real
- Parâmetros internos confidenciais e políticas internas
